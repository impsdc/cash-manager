## Getting Started

### Docker

[Docker documentation](../README.md)

### Local Installation

- Install java jdk-11
- Install Maven
- Install dependencies
`mvn install`
- Setup application.properties with database information and Keys
`cp application.properties.template application.properties`

## Authentication

This application uses JWT authentication.

A JWT is sent when signing in and needs to be sent when attempting to use an authenticated route.
The JWT is used as a Bearer token in the Authorization Header.
